- 미완성 항목: Empty circle + text-muted-foreground

## 3. 협업 제안 Modal 설계 (Minimal & Intuitive)

### Modal 구조

```
┌─────────────────────────────────────────────┐
│  💬 협업 제안하기                 [X]        │
│  김창업님에게 협업을 제안해보세요.           │
│  구체적일수록 긍정적인 답변을 받을 확률이     │
│  높아요.                                    │
│                                             │
│  제안 주제                                  │
│  ┌─────────────────────────────────────┐   │
│  │ 예: AI 기반 SaaS 공동 개발 제안     │   │
│  └─────────────────────────────────────┘   │
│                                             │
│  메시지                                     │
│  ┌─────────────────────────────────────┐   │
│  │ 구체적인 협업 내용, 제안 배경,      │   │
│  │ 기대하는 시너지 등을 작성해주세요.   │   │
│  │                                     │   │
│  │                                     │   │
│  │                                     │   │
│  └─────────────────────────────────────┘   │
│                                             │
│  연락처                                     │
│  ┌─────────────────────────────────────┐   │
│  │ your.email@example.com              │   │
│  └─────────────────────────────────────┘   │
│                                             │
│  [취소]            [전송하기]               │
└─────────────────────────────────────────────┘
```

### Modal 디자인 원칙

1. **최소 입력 필드**
   - 제안 주제 (1줄)
   - 메시지 (6줄 Textarea)
   - 연락처 (1줄)
   - 총 3개 필드만 (이탈 방지)

2. **Placeholder 가이드**
   - 예시 포함 ("예: AI 기반...")
   - 어떻게 작성할지 명확히 안내
   - 진입 장벽 낮춤

3. **토스 스타일 인터랙션**
   - Background: bg-card
   - Border: border-border/50
   - Border-radius: rounded-3xl (24px)
   - Input height: 48px (h-12)
   - Input border-radius: rounded-2xl (16px)
   - Button height: 48px (h-12)

4. **긍정적 메시지**
   - "구체적일수록 긍정적인 답변을..."
   - 성공 확률 상승 암시
   - 작성 동기 부여

## 4. 비즈니스 협업 전환 플로우

### 전환 단계 (Conversion Funnel)

```
1. 프로필 발견
   ↓
   [지원사업 큐레이션, 쓰레드, 아티클 등에서 프로필 카드 노출]
   - 프로필 완성도 배지 표시
   - 전문 분야 칩 미리보기
   - "협업 가능" 인디케이터

2. 프로필 클릭
   ↓
   [상세 프로필 진입]
   - 프로필 완성도 85% 배너 (자기 프로필인 경우)
   - Premium Business Card 느낌
   - 신뢰도 지표 (창업 경험, 투자 단계, 네트워크)

3. 신뢰도 구축
   ↓
   [전문 분야 & 이력 확인]
   - #00E5FF 칩으로 전문성 시각화
   - 타임라인으로 경력 검증
   - Achievement로 성과 확인

4. 니즈 매칭
   ↓
   [협업 니즈 섹션 확인]
   - "나랑 맞네!" 순간
   - 글로우 효과로 시선 집중
   - 구체적 설명으로 확신

5. 액션 결정
   ↓
   [고정 CTA 버튼]
   - 가벼운 시작: 커피챗 신청
   - 명확한 목적: 협업 제안하기

6. 제안 작성
   ↓
   [Modal 입력]
   - 최소 필드 (3개)
   - 가이드 포함 Placeholder
   - 긍정적 메시지

7. 제안 전송
   ↓
   [알림 & 매칭]
   - 상대방에게 알림 전송
   - 프로필 상호 열람
   - 채팅 or 이메일로 연결

8. 비즈니스 협업 시작 🎉
```

## 6. 프로필 → 비즈니스 기회 전환 지표

### 추적할 메트릭 (Metrics)

1. **프로필 조회수** (Profile Views)
   - 얼마나 발견되는가?

2. **협업 제안 전환율** (Proposal Rate)
   - 프로필 조회 → 제안 버튼 클릭 비율
   - 목표: 15% 이상

3. **제안 수락률** (Acceptance Rate)
   - 제안 전송 → 긍정 답변 비율
   - 목표: 30% 이상

4. **프로필 완성도 vs 제안 수 상관관계**
   - 완성도 높을수록 제안 많이 받는지

5. **협업 니즈 작성 여부 vs 제안 수**
